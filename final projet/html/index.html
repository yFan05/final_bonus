智能理財機器人
Group11 李宜熹 謝沛璇 楊凡萱 莊紫喬 江旻柔

基本介紹

理財機器人主要是透過演算法（ALGORITHM）、AI 等技術， 並完全經由網路互動輔以『全無』或『極少』的人工服務，以自動化工具向客戶提供投資組合建議的顧問服務（ROBO-ADVISOR），又稱之自動化投資顧問服務。

．發展背景

由於金融海嘯使得大眾對銀行失去信心， 而一般人多少都要接觸到投資理財的部分，企業和各種金融機構平常更是需要進行一些投資決策、風險管理等，顧自2008年起逐步發展出機器人投資顧問

．優勢


- 有效降低風險
- 提供客觀判斷
- 利益不衝突
- 較低理財門檻

．劣勢


- 無法預測突發狀況
- 出錯時之責任歸屬
- 法律規範不完善

機器模型

迴歸模型

預測連續數值，如市場走勢、股票價格

分類模型

預測離散數值，將資料分到指定的類別中，如風險評估、資產分配

時間序列分析模型

分析時間序列的資料之規律性和趨勢，進而預測未來動向

自然語言處理模型(NLP)

讓電腦能解譯、理解人類語言

推薦系統

用數據分析和機器學習技術提供個性化投資建議

集成學習

透過多個模型組合來解決單一預測問題，如隨機森林、梯度提升樹

問題與解決辦法

（一）無法有效衡量可能風險

人工智慧潛在風險不同於以往的模式， 且尚未完全知悉，故迄今仍無有效的方法可 以評估人中能對於金融體系的影響性。

．風險管理模型
1. VAR 模型
衡量投資組合在特定信賴水平下可能損 失的統計方法
2.CVAR 模型
CVAR 提供超過 VAR 水平的平均損失
3. 壓力測試模型
通過對投資組合施加不同的市場情境和 壓力測試，評估不同市場條件下的表現
4. 模擬模型
使用模擬方法生成大量可能的市場情境， 並評估每種情境下投資組合的損益

（二）難以套用既有監管經驗

智能理財助理牽扯到科技倫理範圍，使用者須考慮隱私權、著作權、資安風險等。 現今對於數位理財的法律尚未完備，欠缺合適的規制而難以套用既有的理財經驗。

．現行適用法規
1. 證券投資信託及顧問法
2. 金融消費者保護法
3. ROBO-ADVISOR作業要點

．法規發展方向
1. 監管者須熟悉智能理財背後原理
2. 修法增訂機器人理財的重要性原則
3. 授權投信公會成立審查小組
4. 著重演算法適用性評估衡
5. 告知顧客自動化投資須注意之事項

現代應用

Betterment

在金融海嘯後成立，目前被認為是市場上第一家 機器人理財公司。根據客戶所填入的資料建議適 合的投資策略，然後透過其演算法「客製化」包 含多種股票、債券和基金的投資組合。

．沒有限制客戶的最低開戶金額及投資金額
．分析用戶的行為，給出更好的投資行為建議
．主打退休理財

Alpha

根據投資人的財務情況和風險承受度，量身打資產 配置建議，並且透過演算法模擬成功達成投資目標 的機率有多少。一旦確定投資計畫，後續皆由機器 人進行投資和調整。當資產比例偏離過大或是每年 一次進行再平衡。
